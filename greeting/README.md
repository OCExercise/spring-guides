# greeting

Greeting service

## Usage

### Gradle
```
$ gradle bootRun
....
# Browse to http://localhost:8080/greeting -> "Hello, World"
# Browse to http://localhost:8080/greeting?name=Prez%20Cannady -> "Hello, Prez Cannady"
# Browse to http://localhost:8080/greeting/Prez%20Cannady -> "Hello, Prez Cannady"
```
